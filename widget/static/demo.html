Встраиваемый виджет RAG-ассистента

На этой странице подключён виджет в правом нижнем углу. Он обращается к API на том же хосте (/api/chat).

Встраивание на свой сайт

Скопируйте фрагмент ниже, подставив URL вашего сервера виджета:

<script
  src="https://ВАШ-ДОМЕН/widget/embed.js"
  data-api-url="https://ВАШ-ДОМЕН"
  data-title="Поддержка"
  data-primary-color="#2563eb"
></script>

Доступные атрибуты data-*:

  • data-api-url — базовый URL API (по умолчанию — origin скрипта)
  • data-assets-base — базовый URL статики, если отличается от API
  • data-title, data-subtitle, data-welcome — тексты интерфейса
  • data-primary-color — основной цвет (HEX)
